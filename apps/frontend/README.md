# Frontend — Next.js

El frontend utiliza una arquitectura basada en características.

## Directorios

- `app`: rutas, páginas y layouts.
- `features`: funcionalidades del negocio.
- `shared`: componentes reutilizables.
- `core`: autenticación, configuración y acceso a la API.
- `styles`: estilos globales.

Los archivos `.tsx` y `.ts` están vacíos porque esta entrega define únicamente la arquitectura.
