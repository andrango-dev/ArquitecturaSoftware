# Casos de uso principales

## Administrador

- Gestionar usuarios y roles.
- Configurar permisos.
- Consultar reportes.
- Revisar auditorías.

## Recepcionista

- Registrar pacientes.
- Actualizar datos personales.
- Agendar, reprogramar y cancelar citas.
- Consultar disponibilidad médica.

## Médico

- Consultar agenda.
- Registrar consulta.
- Registrar diagnóstico.
- Emitir receta.
- Solicitar exámenes.
- Consultar historia clínica.

## Enfermero

- Registrar signos vitales.
- Actualizar seguimiento del paciente.
- Apoyar hospitalización.

## Laboratorista

- Consultar solicitudes.
- Registrar resultados.
- Validar resultados.

## Farmacéutico

- Gestionar medicamentos.
- Dispensar recetas.
- Controlar inventario.

## Cajero

- Generar factura.
- Registrar pago.
- Consultar saldos.

## Paciente

- Consultar citas.
- Consultar recetas.
- Consultar resultados autorizados.
