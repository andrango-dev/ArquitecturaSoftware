# Documentación de arquitectura

Este directorio reúne la documentación del sistema hospitalario.

- `DECISIONES_ARQUITECTONICAS.md`: decisiones y justificaciones.

- `MODELO_DATOS.md`: entidades propuestas.
- `SEGURIDAD.md`: autenticación, autorización y auditoría.
